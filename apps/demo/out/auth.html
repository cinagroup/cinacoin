⚡ Implement SIWE in 3 Lines

CinaConnect handles the complexity. You write the UX.

import { SiweMessage } from 'siwe'; import { CinaConnect } from '@cinaconnect/sdk'; const client = new CinaConnect({ chains: ['ethereum', 'polygon'] }); const message = new SiweMessage({ domain: window.location.host, address, ... }); const signature = await client.wallet.signMessage(message.prepareMessage()); const verified = await client.auth.verify({ message, signature });
